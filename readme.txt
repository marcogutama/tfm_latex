https://gestor.unir.net/userFiles/file/guias/visual12_guia.pdf

Para utilizar esta plantilla:
1. Rellenar los campos de entrada en TFM.tex: título, autor, máster, etc.
2. Incluir tantos archivos de texto como capítulos en la carpeta "contenido" y agregarlos a la lista de capítulos a incluir (dentro TFM.tex). Para ello se puede utilizar el capítolo X "TFM-04_Capitulo_X.tex" como plantilla para los demás capítulos.
3. Incluir en el archivo "TFM-biblio.bib" la bibliografía en formato BibTex. 
4. Se recomienda los siguientes compiladores de Latex de Overleaf:
   - pdfLatex: compilación más rápida; fuentes TTF convertidas con mapas TFM*
   - XeLatex: pdf más optimizado (pequeño) pero compilación más lenta.
  *La calidad de las fuentes TFM es más baja que la de las fuentes TTF.
5. Se recomiendo utilizar pdfLatex en modo borrador y XeLatex en caso contrario.
